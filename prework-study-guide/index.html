Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Quick Notes:
  • h: Heading; H1 is largest
  • p: Paragraph
  • ui: Unordered Lists
  • oi: Ordered Lists
  • li: Liine Items
  • img: Image
  • src: Points to location of image
  • br within brackets: line or page break
  • a within brackets: anchor
  • href: points to a URL

CSS

  • CSS: Cascading Style Sheets
  • CSS is a style sheet language used for describing the presentation of a doc written in HTML or XML
  • CSS is basically what makes the page pretty.

Git

  • GIT: Global Information Tracker for coding
  • Control system that tracks changes in files, usually for coordinating work among other programmers collaboratively
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a programming language used to create interactive effects within web browsers
